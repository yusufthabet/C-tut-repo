# C-tut-repo
c# tutorial repo

## Hello World Program

This repository contains a simple C# console application that prints "world".

### Prerequisites
- .NET SDK (version 9.0 or later)

### Building the Program
Run the following command to build the project:
```
dotnet build
```

### Running the Program
To run the program:
```
dotnet run
```
or
```
dotnet bin/Debug/net9.0/HelloWorld.dll
```

### Expected Output
```
world
```

### Troubleshooting
- If you encounter a framework version error, ensure you have the correct .NET runtime installed.
- Update the TargetFramework in HelloWorld.csproj if needed.
